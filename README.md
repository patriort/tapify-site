# tapify-site

Pequeña landing para Tapify — scaffold Next.js.

Instrucciones locales:

```bash
# instalar dependencias
npm install

# modo desarrollo
npm run dev

# build
npm run build
npm start
```

Desplegar en Vercel:
1. Conecta tu cuenta de GitHub a Vercel.
2. Importa el repo `tapify-site` y dale deploy.

Si quieres, puedo conectar y desplegar el repo por ti (necesitaré acceso a GitHub o que me proporciones el repositorio remoto ya creado).
